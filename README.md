# NYU Machine Learning Projects

Two collaborative machine learning projects demonstrating image classification and regression analysis.

## Projects

### 1. CIFAR-10 ResNet Classification
Image classification using transfer learning with ResNet50V2.
- **Accuracy**: Tracked over 11 epochs
- **Framework**: TensorFlow/Keras
- **Dataset**: CIFAR-10 (50,000 training + 10,000 test images)
- [Presentation PDF](./cifar10-resnet-classification/presentations/CIFAR10%20ResNet%20Classification.pdf) | [PPTX](./cifar10-resnet-classification/presentations/CIFAR10%20ResNet%20Classification%20Presentation.pptx)
- [Notebook](./cifar10-resnet-classification/notebooks/CIFAR10_ResNet_Classification.ipynb)

### 2. Fish Market Model & Loss Function Analysis
Regression analysis comparing different loss functions for fish weight prediction.
- **Framework**: TensorFlow/Keras
- [Presentation PDF](./fish-market-analysis/presentations/Fish%20Market-Model%20and%20Loss%20Function%20Analysis.pdf) | [PPTX](./fish-market-analysis/presentations/Fish%20Market%20Model%20and%20Loss%20Functions%20Data%20Presentation.pptx)
- [Notebook](./fish-market-analysis/notebooks/Fish_Market_Model_Loss_Function_Analysis.ipynb)
